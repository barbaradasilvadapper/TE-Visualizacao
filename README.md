---

## 📌 Sobre o Projeto

Este projeto foi desenvolvido na disciplina de Visualização de Dados, com o objetivo de criar soluções interativas que auxiliem a análise e a tomada de decisão da Fundação Pão dos Pobres.

A proposta busca integrar e explorar dados relacionados aos programas de acolhimento, aprendizagem profissional e serviços de convivência da instituição, utilizando técnicas de tratamento, organização e visualização de dados.

---

## 🎯 Objetivos

- Entender e organizar as planilhas fornecidas;
- Identificar indicadores relevantes para análise dos programas sociais;
- Aplicar técnicas de limpeza, transformação e integração de dados;
- Desenvolver dashboards interativos para análises temporais e comparativas;
- Apoiar a interpretação dos resultados obtidos;
- Promover a aplicação prática de conceitos de visualização de dados em um contexto real.
